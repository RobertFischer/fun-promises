gracefulify

  • gracefulify<T>(fsModule: T): T
    • Defined in node_modules/@types/graceful-fs/index.d.ts:19

    Use this method to patch the global fs module (or any other fs-like module). NOTE: This should only ever be done at the top-level application layer, in order to delay on EMFILE errors from any fs-using dependencies. You should not do this in a library, because it can cause unexpected delays in other parts of the program.

    Type parameters

    • T

    Parameters

    • fsModule: T

      The reference to the fs module or an fs-like module.

    Returns T
